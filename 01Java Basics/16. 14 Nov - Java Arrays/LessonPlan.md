# Array 
```java
- Array is data structure to store collection of homogenous data in sequentially manner inside memory.

- an array is 0-based indexing.

- The memory allocation in array is contiguous.

- An array can be single or multi dimentional based on utility.
```

# Array Declaration and Creation
```java
#Declarization

- Syntax  ---> <dataType> <arrayName>[]; OR <dataType>[] <arrayName>;

- Example ---> String cars[];            OR String[] cars;

#Initialization

- Syntax  ---> <dataType>[] <arrayName> = new <dataType>[size];

- Example ---> int[] rollNo = new int[20]; OR int[] rollNo = {1,2,3,4};
```

# Types of Array
```java
1. Single Dimentional     
2. Multi-dimentional

1.Single dimentional
    int[] ages = new ages[4]

2. Multidimentional Arrays
    int[][] corona = {{1,2,3},
                      {3,4,5},
                      {5,7,3}};
```
